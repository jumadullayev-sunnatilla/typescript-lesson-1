TypeScript Haqida

TypeScript bu JavaScript-ning kengaytirilgan versiyasi bo‘lib, katta loyihalarda ishlashni osonlashtiradi.

Asosiy Imkoniyatlar

TypeScript quyidagilarni taklif qiladi:

1. Statik tiplash

2. Sinflar va interfeyslar

3. Kodni boshqarishning osonlashuvi

Misol

TypeScript kodidan oddiy misol:

const message: string = "Salom, TypeScript!";

console.log(message);

Qanday Ishlatiladi?

TypeScript-ni o‘rnatish uchun quyidagini ishlating:

npm install -g typescript

Keyin esa TypeScript fayllarini JavaScript-ga o‘tkazish uchun:

tsc fayl_nomi.ts
